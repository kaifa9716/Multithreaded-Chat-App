Instructions :
1. Navigate to the file directory.
2. Run in terminal :  
  javac ServerSide.java ChatClient.java  
  java ServerSide
3. Go to another terminal/ or start a new terminal.
  then, 
  java ChatClient
4. Enter name of the client
5. Join more clients thourgh multiple terminal and chat
